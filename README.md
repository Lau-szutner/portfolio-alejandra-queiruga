# Professional Portfolio – Criminal Law Attorney

This repository contains the source code for the professional portfolio website of **Alejandra Queiruga**, a specialized criminal law attorney. The website was designed to showcase her expertise, experience, and legal services through a modern, elegant, and responsive interface.

## 🚀 Tech Stack

- **Next.js** – React framework for server-side rendering and optimized performance
- **React** – Component-based UI library for dynamic, interactive interfaces
- **TypeScript** – Typed JavaScript for improved code quality and maintainability
- **CSS Modules / Styled Components / Tailwind CSS** – (based on the chosen styling method)
- **ESLint & Prettier** – Code linting and formatting tools for consistent development

## 🎯 Purpose

The goal of this project is to create a digital presence that reflects the professionalism and legal expertise of the attorney. It features sections such as:

- Introduction and personal background
- Areas of legal specialization (Criminal Law)
- Services offered
- Contact form and professional links
- Responsive design for mobile and desktop
